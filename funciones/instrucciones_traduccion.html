Instrucciones para Implementar Traducción en Todas las Páginas

Paso 1: Asegurarse que el archivo translate.js existe

Ya has creado el archivo PI2do/translate.js con todo el código necesario para la traducción.

Paso 2: Agregar a cada archivo HTML

Para cada archivo HTML de tu sitio, agrega lo siguiente dentro de la sección <head>:

<!-- Google Cloud Translation API -->
<script src="https://www.gstatic.com/firebasejs/9.6.10/firebase-app-compat.js"></script>
<script src="https://www.gstatic.com/firebasejs/9.6.10/firebase-auth-compat.js"></script>
<!-- Script de traducción global -->
<script src="/PI2do/translate.js"></script>

Paso 3: Asegúrate que la estructura HTML de los botones de idioma sea correcta

En cada página, asegúrate de tener esta estructura en el HTML para los botones de idioma:

<!-- Bandera actual -->
<div id="idiomaToggle">
    <img class="españa" id="banderaIdioma" src="/PI2do/imagenes/logos/espanol.png" alt="Idioma">
</div>

<!-- Opción secundaria -->
<div id="idiomasOpciones">
    <img class="ingles" src="/PI2do/imagenes/logos/ingles.png" alt="Cambiar a inglés">
    <img class="españa" src="/PI2do/imagenes/logos/espanol.png" alt="Cambiar a español">
</div>

Paso 4: Eliminar código JavaScript duplicado

Si ya tienes código JavaScript relacionado con el cambio de idioma en tus archivos HTML, puedes eliminarlo ya que ahora está en translate.js. Solo mantén el código para mostrar/ocultar el menú de opciones de idioma si lo tienes separado.

Paso 5: Excluir elementos de la traducción (opcional)

Si tienes elementos que no quieres que se traduzcan (como nombres de marca, términos técnicos, etc.), puedes agregarles el atributo data-no-translate:

<span data-no-translate>PRODCONS</span>

Importante

Recuerda que el API de Google Translation tiene un límite de uso y puede generar costos después de cierto volumen de traducciones. Asegúrate de monitorear su uso.

Ejemplo completo para una página

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>PRODCONS</title>
    <!-- Tus estilos CSS existentes -->
    <link rel="stylesheet" href="/PI2do/styles.css">
    
    <!-- Google Cloud Translation API -->
    <script src="https://www.gstatic.com/firebasejs/9.6.10/firebase-app-compat.js"></script>
    <script src="https://www.gstatic.com/firebasejs/9.6.10/firebase-auth-compat.js"></script>
    <!-- Script de traducción global -->
    <script src="/PI2do/translate.js"></script>
</head>
<body>
    <!-- Tu contenido HTML existente -->
    
    <!-- Banderas para cambio de idioma -->
    <div id="idiomaToggle">
        <img class="españa" id="banderaIdioma" src="/PI2do/imagenes/logos/espanol.png" alt="Idioma">
    </div>
    <div id="idiomasOpciones">
        <img class="ingles" src="/PI2do/imagenes/logos/ingles.png" alt="Cambiar a inglés">
        <img class="españa" src="/PI2do/imagenes/logos/espanol.png" alt="Cambiar a español">
    </div>
    
    <!-- Resto de tu contenido -->
</body>
</html>
